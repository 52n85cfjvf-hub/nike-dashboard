# Дашборд продаж NIKE

**Динамический одностраничный дашборд квартальных продаж NIKE Brand**

> Когда, какой продукт и в каком регионе NIKE продаётся лучше,
> и какие направления могут быть приоритетными в следующих кварталах?

---

## Быстрый старт

```bash
# Установить зависимости
pip install -r requirements.txt

# Запустить дашборд
streamlit run app.py
```

---

## Бизнес-вопрос

Дашборд позволяет ответить на вопросы:
- В каком квартале продажи исторически выше?
- Какой продукт лучше продаётся в каждом регионе?
- В каком регионе лучше продаётся каждый продукт?
- Какие сочетания «продукт × регион» наиболее перспективны?
- Каков прогноз выручки и доли на 4 квартала вперёд?

---

## Данные

**Источники:** 15 квартальных пресс-релизов NIKE (PDF) + 3 годовых 10-K отчёта

**Период:** Q1 FY2023 — Q1 FY2026 (13 кварталов)

**Детализация:** 3 продукта × 4 региона × 13 кварталов = 156 наблюдений

| Категория | Значения |
|-----------|----------|
| Продукты | Обувь (Footwear), Одежда (Apparel), Экипировка (Equipment) |
| Регионы | Северная Америка, EMEA, Большой Китай, APLA |
| Метрика | Квартальная выручка NIKE Brand, млн $ США |

> Converse, Corporate и Global Brand Divisions исключены из анализа.

## Структура дашборда

### Фильтры (в шапке)
- **Период** — выбор начального и конечного квартала
- **Продукт** — мультивыбор: Обувь / Одежда / Экипировка
- **Регион** — мультивыбор: 4 региона

### KPI-карточки
- Выручка выбранного среза (последний квартал)
- Рост год к году
- Доля в анализируемой выручке NIKE Brand
- Лучший квартал по средней исторической выручке

### Блок 1 — Динамика, сезонность и прогноз
- График выручки: факт (сплошная) + прогноз (пунктир) на 4 квартала
- График доли: факт + прогноз
- Столбчатая диаграмма средней выручки по Q1–Q4
- Таблица сезонности с лучшим продуктом и регионом для каждого квартала
- Автоматический текстовый вывод о сезонности

### Блок 2 — Карта продаж
- Тепловая карта выручки: регионы × продукты
- Автоматический вывод о лидере

### Блок 3 — Приоритетные направления
- Пузырьковая диаграмма: X=выручка, Y=рост г/г, размер=доля
- Зоны: Приоритетные / Зрелые / Развивающиеся / Слабые
- Автоматические выводы о лидерах

---

## Прогнозы

**Модель:** Holt-Winters (ExponentialSmoothing)
- Аддитивный тренд + аддитивная сезонность
- Сезонный период: 4 квартала
- Горизонт: 4 квартала

**Резервный метод:** Сезонный наивный (прогноз = факт года назад)

**Прогноз прогнозируется для каждого из 12 направлений отдельно**, затем агрегируется:
- Прогноз выручки = сумма прогнозов выбранных направлений
- Прогноз доли = прогноз выручки / прогноз общей выручки × 100%

## Ограничения

- Прогнозы носят учебный характер
- Доступно только 13 кварталов истории
- Анализируется выручка, а не прибыль
- Валютные курсы не корректируются
